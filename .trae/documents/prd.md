## 1. 产品概述

面向品牌公关团队的桌面端敏感词复盘工具，用于新品发布、代言人事件或售后危机期间的舆情告警事后分析。公关经理将一段时间内的命中记录导入后，进行词组归因、事件时间线梳理和复盘报告生成，适合企业公关会议前快速整理材料。

- 核心用户：品牌公关经理、公关总监、危机应对团队成员
- 核心价值：将零散的敏感词命中数据转化为结构化归因和可视化复盘报告，提升公关会议准备效率

## 2. 核心功能

### 2.1 用户角色

| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 公关经理 | 直接使用 | 导入数据、词组归因、拖拽合并、查看时间线、生成报告 |
| 公关总监 | 直接使用 | 查看报告、审批关注清单、导出报告 |

### 2.2 功能模块

1. **数据导入页**：CSV/Excel 文件上传，字段映射，数据预览与清洗
2. **词组归因页**：按标签分类敏感词，拖拽合并近义表达，词频统计
3. **事件时间线页**：敏感词激增时间轴，官方回应/媒体报道/达人转发节点备注
4. **复盘报告页**：高频敏感词排行、扩散渠道分析、处置耗时统计、后续关注词清单

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 数据导入页 | 文件上传区 | 支持 CSV/Excel 拖拽上传，自动解析字段 |
| 数据导入页 | 字段映射 | 将文件列映射到敏感词、时间、来源、渠道等标准字段 |
| 数据导入页 | 数据预览 | 显示前 50 条记录预览，标记异常数据 |
| 词组归因页 | 标签分类面板 | 按辱骂、抵制、质量质疑、虚假宣传、价格争议等标签展示命中词 |
| 词组归因页 | 拖拽合并区 | 用户可拖拽近义词合并为一个词组，合并后显示总命中次数 |
| 词组归因页 | 词频统计 | 每个标签下的词频柱状图和词云 |
| 事件时间线页 | 时间轴主体 | 横向时间轴，标注敏感词激增点（柱状高度表示激增幅度） |

| 复盘报告页 | 关注词清单 | 可编辑的后续需关注词列表，支持增删和优先级标记 |

## 3. 核心流程

用户打开工具 → 导入敏感词命中记录（CSV/Excel）→ 系统解析并映射字段 → 进入词组归因页，按标签浏览命中词，拖拽合并近义表达 → 切换到事件时间线，查看敏感词激增节点，添加事件备注 → 生成复盘报告，查看高频词、渠道分布、处置耗时，编辑关注词清单 → 导出报告用于公关会议

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深灰（#1A1D23）为底，搭配警示橙（#F97316）作为强调色，辅以冷蓝（#3B82F6）作为信息色
- **次色调**：标签使用差异化配色——辱骂红(#EF4444)、抵制紫(#8B5CF6)、质量质疑黄(#EAB308)、虚假宣传橙(#F97316)、价格争议蓝(#3B82F6)
- **按钮风格**：圆角矩形（8px），轻微阴影，hover 时提升阴影层级
- **字体**：标题使用 DM Sans（粗体），正文使用 Noto Sans SC，数据使用 JetBrains Mono
- **布局风格**：左侧导航栏 + 右侧内容区，卡片式布局，表格数据驱动
- **图标**：使用 Lucide Icons 线性图标风格

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 数据导入页 | 文件上传区 | 虚线边框拖拽区，上传图标，支持文件列表显示 |
| 数据导入页 | 字段映射 | 左侧文件列名 → 右侧标准字段下拉选择 |
| 数据导入页 | 数据预览 | 条纹表格，异常数据高亮行 |
| 词组归因页 | 标签分类面板 | 左侧标签切换标签页，右侧词列表带命中次数徽章 |
| 词组归因页 | 拖拽合并区 | 可拖拽词卡片，合并时显示吸附动画，合并后显示合并标签 |
| 词组归因页 | 词频统计 | 水平柱状图 + 迷你词云 |
| 事件时间线页 | 时间轴主体 | 横向滚动时间轴，激增点用竖向柱状表示，hover 显示详情浮层 |
| 事件时间线页 | 节点备注 | 点击激增点弹出侧面板，可添加不同类型备注卡片 |
| 事件时间线页 | 筛选控制 | 顶部筛选条，标签多选、渠道多选、日期范围选择器 |
| 复盘报告页 | 高频敏感词 | 横向条形图 Top 20，右侧迷你趋势折线 |
| 复盘报告页 | 扩散渠道分析 | 环形饼图 + 图例列表 |
| 复盘报告页 | 处置耗时 | 三个统计卡片（平均/最长/最短），配合柱状对比图 |
| 复盘报告页 | 关注词清单 | 可编辑表格行，优先级标签（高/中/低），增删按钮 |

### 4.3 响应式设计

- 桌面优先设计，最小宽度 1280px
- 左侧固定导航栏（240px），右侧内容区自适应
- 时间线支持横向滚动以适应大量事件节点
- 报告页支持打印友好的布局
